# Project Amethyst

Project Amethyst is an open source-project for building client-side mods for Minecraft Bedrock `1.20.51.1`, client-side mods are able to stretch past the limitations of addons and have full control over the game such as using Minecraft's input system, creating your own user interfaces, changing vanilla functionality, and more.
